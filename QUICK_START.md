## Get Started in 3 Steps!

### Step 1: Install & Run
```bash
npm install
npm run dev
```

### Step 2: Add Your Photos
**Choose one method:**

**Method A: Use the batch file (Easiest!)**
- Double-click `open-images-folder.bat`
- The images folder will open automatically
- Copy your 10 photos there

**Method B: Manual**
- Navigate to `public/images/` folder
- Copy your photos there

### Step 3: Name Your Photos
Rename your photos to:
- **Gallery**: `photo1.jpg`, `photo2.jpg`, `photo3.jpg`, `photo4.jpg`, `photo5.jpg`, `photo6.jpg`
- **Memories**: `memory1.jpg`, `memory2.jpg`, `memory3.jpg`, `memory4.jpg`

Then refresh your browser!
